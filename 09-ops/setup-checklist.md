# Setup Checklist

> Complete this checklist when setting up the PM AI OS in a new environment.

## GitHub

- [ ] pm-ai-skills repo created (private)
- [ ] pm-ai-os repo created (private)
- [ ] pm-ai-demo repo created (private)
- [ ] All repos cloned locally
- [ ] Git user config set (`git config --global user.name` and `user.email`)

## Claude

- [ ] Claude.ai access confirmed
- [ ] Claude Code installed (if using local dev workflow)
- [ ] CLAUDE.md files reviewed in pm-ai-os and pm-ai-demo

## NotebookLM

- [ ] NotebookLM account access confirmed
- [ ] First notebook created per notebook-structure.md
- [ ] Source tagging convention reviewed

## Automation (when ready)

- [ ] Relay / Make / Zapier account configured
- [ ] MCP servers documented in mcp-setup.md
- [ ] Connector map reviewed and up to date

## Ongoing

- [ ] Weekly loop operational (see 02-workflows/weekly-pm-loop.md)
- [ ] At least one skill tested end-to-end
- [ ] First eval run on an AI output
