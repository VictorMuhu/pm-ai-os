# Demo Eval

> Evaluation rubric for demo materials in pm-ai-demo.

## Dimensions

| Dimension | Score (1–5) | Notes |
|---|---|---|
| Sanitization | | Are all real names, metrics, and confidential details removed? |
| Realism | | Do examples feel realistic and credible, not toy-like? |
| Clarity | | Is the narrative clear to someone unfamiliar with the context? |
| Demoability | | Can this be walked through live in the stated time? |
| Executive readiness | | Is the output polished and credible for a senior audience? |
| Fictional integrity | | Are fictional details internally consistent? |

## Readiness rating

- **Ready**: All dimensions 4+, sanitization confirmed
- **Needs revision**: Any real data present = automatic redo regardless of other scores

## Common failure modes

- Real customer names or metrics left in place
- Examples that are too abstract to be convincing
- Narrative too long or technical for the demo slot
- Internal inconsistencies in fictional data
