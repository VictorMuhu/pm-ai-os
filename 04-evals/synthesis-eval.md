# Synthesis Eval

> Evaluation rubric for AI-generated customer research synthesis.

## Dimensions

| Dimension | Score (1–5) | Notes |
|---|---|---|
| Evidence grounding | | Are all insights backed by at least 2 sources? |
| Segmentation | | Is the segment explicit (not just "customers")? |
| Pattern quality | | Are patterns distinct and non-overlapping? |
| Tension identification | | Are contradictions surfaced rather than smoothed over? |
| Gap identification | | Are thin-evidence areas flagged? |
| Fabrication check | | Are any quotes invented or paraphrased without attribution? |

## Readiness rating

- **Ready**: All dimensions 4+, no fabricated evidence
- **Needs revision**: Any fabricated evidence = automatic redo; otherwise apply standard rubric

## Common failure modes

- Insights stated without evidence citations
- All segments lumped together
- Fabricated or paraphrased quotes presented as verbatim
- Gaps hidden rather than disclosed
