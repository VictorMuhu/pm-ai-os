# Quarterly PM Loop

> Starter file. Define your quarterly AI-assisted PM planning routine here.

## Purpose

A deep quarterly planning cycle that connects customer evidence to strategy, produces a prioritized roadmap, and aligns stakeholders on goals for the next 3 months.

## Inputs

- OKRs or strategic goals for the quarter
- Accumulated customer research and feedback
- Previous quarter retrospective
- Competitive and market signals

## Loop steps

1. **Research consolidation**: Run Customer Synthesis skill across all Q evidence
2. **Opportunity mapping**: Run Opportunity Solution Tree skill to map the problem space
3. **Problem framing**: Run Problem Framing skill for top 2–3 initiatives
4. **Prioritization**: Run Prioritization skill to rank initiatives against OKRs
5. **Roadmap narrative**: Run Roadmap Narrative skill to write the strategic narrative
6. **Metric stack review**: Run Metric Stack skill to confirm metrics align with new goals
7. **Stakeholder alignment**: Present roadmap and narrative to leadership

## AI touchpoints

- Customer Synthesis, OST, Problem Framing, Prioritization, Roadmap Narrative, Metric Stack skills

## Notes

- Quarterly loop should produce: research synthesis, prioritized roadmap, narrative doc, and updated metric stack.
- Plan 2–3 weeks for this cycle, not a single session.
